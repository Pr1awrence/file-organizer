This is a simple Python script to organize files by their type into categorized folders.

<!-- GETTING STARTED -->
## Getting Started
### Prerequisites
Install npm
* npm
  ```sh
  npm install npm@latest -g
  ```
  
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/Pr1awrence/file-organizer.git
   ```
2. Provide a directory in cmd
   ```sh
   python organizer.py --directory=C:\pathtoyourfolder\
   ```
